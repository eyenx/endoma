Startseite

Die Startseite von EnDoMa ist nicht mehr als eine reine Willkommensseite. Kurz und prägnant wird die Applikation erklärt und auf die Loginseite verwiesen.

Loginseite

Auf der Loginseite wird der Benutzer nach Benutzername und Passwort angefragt. Ein erfolgreiches Login leitet der Benutzer direkt auf das Dashboard um. Falls es zu einem fehlerhaftes Login kommt, wird dies entsprechend angezeigt.

Dashboard

Beim Dashboard werden die vier Hauptmenüs kurz vorgestellt. Der Benutzer kann über den entsprechenden Link in das gewünschte Untermenü zugreiffen.

Verwaltung Docker-Hosts

Unter Docker-Hosts erreicht man ist eine Auflistung der hinzugefügten Docker-Hosts des eingeloggten Benutzers. Hier kann man sich die Details eines Hosts über das Auge-Symbol anzeigen lassen oder ein neues Host über "Host hinzufügen" in der Liste aufnehmen.

Die Hostliste kann mittels des Filterkästchens nach Wunsch filtriert werden

Docker-Host hinzufügen

Beim Hinzufügen eines Hosts muss mindestens ein Name definiert werden. Der Benutzer wird jedenfalls hingewiesen, falls nicht alle erforderlichen Felder ausgefüllt wurden.

Mit ein Klick auf das grüne Plus-Symbol wird der Host hinzugefügt.

Die Liste wird sofort aktualisiert und der neu hinzugefügte Docker-Host ist darunter aufzufinden.

Detailansicht Docker-Host

Die Detailansicht eines Hosts erreicht man über das Auge-Symbol auf der Host-Auflistung. Dank dieser Ansicht sind dem Benutzer mehrere Informationen über den Docker-Hosts ersichtlich. Neben Docker-Version, aktueller Status und letzte Aktualsierungszeitpunkt ist auch das generierte API-Key ersichtlich. Diese kann nun dem Benutzer dienen um die korrekte Installation des EnDoMa-Agent gemäss Installationsanleitung vorzunehmen.

Über die Detailansicht kann ein Host auch gelöscht werden. Bei dieser Funktion ist aber Vorsicht zu geniessen, denn bei der Löschung des Hosts werden sich alle darauf befindende Container auch gelöscht.

Verwaltung Docker-Containers

Im Untermenü Docker-Container wird eine Auflistung aller hinzugefügten Containers angezeigt. Per Klick auf das Auge-Symbol kann auf die Detailansicht eines Containers gelangt werden, falls mehrere Informationen erwünscht sind oder der Container gestartet, gestoppt oder gelöscht werden möchte.

Die Containerliste kann mittels des Filterkästchens nach Wunsch filtriert werden

Docker-Container hinzufügen

Ein neuer Container kann mittels "Container hinzufügen" addiert werden. Der Name, die Eingabe des gewünschten Docker-Image und die Auswahl eines Hosts sind Pflicht für die erfolgreiche Erstellung des Containers. Eine Auswahl an Docker-Images ist im Docker Registry zu finden.

Optional kann noch ein Port angegeben werden, welcher dann vom Docker-Host auf das Docker-Container weitergeleitet wird.

Der Benutzer darf auch eine Verknüpfung mit einem anderen, bereits auf dem ausgewählten Host bestehenden Container erstellen. Dies wird erreicht durch Auswahl des gewünschten Container im Dropdown-Feld "Links". Danach kann mittels des blauen "Plus-Zeichen" die Verknüpfung hinzugefügt werden.

Zum Schluss kann noch eine oder mehrere Umgebungsvariablen hinzugefügt werden. Schlüssel und Wert werden in den entsprechenden Felder eingegeben und mittels Klick auf den blauen "Plus-Zeichen" hinzugefügt

Mit dem Betätigen des grünen "Hinzufügen-Taste" wird der Container auf die Datenbank geschrieben und die entsprechenden Aufträge werden erstellt um das gewählte Image auf den Host herunterzuladen und den Container zu erstellten.

Der neue Container erscheint sofort in der Auflistung, bleibt aber, bis es nicht auf dem Host erstellt wurde, im Status "Uknown".

Nachdem die Erstellungsaufträge abgearbeitet wurden, wird der Container zum Status "Stopped" wechseln.

Detailansicht Docker-Container

Mit Klick auf den "Auge-Symbol" gelangt mann zur Detailansicht des Containers. Hier kann der Container auch gestartet, gestoppt oder gelöscht werden.

Docker-Container starten

Durch wählen der Funktion 'starten' wird dem Docker-Host einen Auftrag übergeben um den Container zu starten.

Sobald der Container gestartet werden konnte, rescheint er in der Auflistung und Detailansicht mit dem Status "Running".

Docker-Container oder Docker-Host löschen

Mit der Taste "delete" Können Container oder gesamte Hosts gelöscht werden. Diese werden Zur Entfernung markiert und sind sobald die Löschnung erfolgreich durchgeführt wurde nicht mehr ersichtlich.

Zu Beachten gilt, dass bei der Löschung eines gesamten Hosts, alle dem Host gehörende Docker-Container auch zur Entfernung markiert werden.

Statusverlauf anzeigen

Die Statusverlaufeinträge können mittels des Filterkästchens nach Wunsch filtriert werden

Benutzerkonto

Beim Benutzerkonto Menü sind dem Benutzer seine Kontaktdaten ersichtlich.

Logout

Über das Logout Knopf oben rechts kann sich der Benutzer von EnDoMa ausloggen.
