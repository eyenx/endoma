Startseite

Die Startseite von EnDoMa ist nicht mehr als eine reine Willkommensseite. Kurz und prägnant wird die Applikation erklärt und auf die Loginseite verwiesen.

Loginseite

Auf der Loginseite wird der Benutzer nach Benutzername und Passwort gefragt. Ein erfolgreiches Login leitet den Benutzer direkt auf das Dashboard um. Falls es zu einem fehlerhaften Login kommt, wird dies entsprechend angezeigt.

Dashboard

Beim Dashboard werden die vier Hauptmenüs kurz vorgestellt. Der Benutzer kann über den entsprechenden Link in das gewünschte Untermenü einsteigen.

Verwaltung Docker-Hosts

Unter Docker-Hosts erhält man eine Auflistung der hinzugefügten Docker-Hosts des eingeloggten Benutzers. Hier kann man sich die Details eines Hosts über das Auge-Symbol anzeigen lassen oder einen neuen Host über "Host hinzufügen" in die Liste aufnehmen.

Die Hostliste kann mittels des Filterkästchens nach Wunsch gefiltert werden.

Docker-Host hinzufügen

Beim Hinzufügen eines Hosts muss mindestens ein Name definiert werden. Der Benutzer wird darauf hingewiesen, falls nicht alle erforderlichen Felder ausgefüllt wurden.

Mit einem Klick auf das grüne Plus-Symbol wird der Host hinzugefügt.

Die Liste wird sofort aktualisiert und der neu hinzugefügte Docker-Host ist am Ende der Liste zu finden.

Detailansicht Docker-Host

Die Detailansicht eines Hosts erreicht man über das Auge-Symbol auf der Host-Auflistung. In dieser Ansicht sind detaillierte Informationen des Docker-Hosts ersichtlich. Neben Docker-Version, aktuellem Status und letztem Aktualisierungszeitpunkt ist auch der generierte API-Key ersichtlich. Dieser kann nun dem Benutzer dienen, um die korrekte Installation des EnDoMa-Agent gemäss Installationsanleitung vorzunehmen.

Über die Detailansicht kann ein Host auch gelöscht werden. Diese Funktion ist aber mit Vorsicht zu geniessen, denn bei der Löschung des Hosts werden alle sich darauf befindenden Container auch gelöscht.

Verwaltung Docker-Containers

Im Untermenü Docker-Containers wird eine Auflistung aller hinzugefügter Container angezeigt. Per Klick auf das Auge-Symbol kann auf die Detailansicht eines Containers gelangt werden, falls mehr Informationen erwünscht sind oder der Container gestartet, gestoppt oder gelöscht werden soll.

Die Containerliste kann mittels des Filterkästchens nach Wunsch filtriert werden.

Docker-Container hinzufügen

Ein neuer Container kann mittels "Container hinzufügen" addiert werden. Der Name, die Eingabe des gewünschten Docker-Image und die Auswahl eines Hosts sind Pflicht für die erfolgreiche Erstellung des Containers. Eine Auswahl an Docker-Images ist im Docker Registry zu finden.

Optional kann ein Port angegeben werden, welcher dann vom Docker-Host auf den Docker-Container weitergeleitet wird.

Der Benutzer darf auch eine Verknüpfung mit einem anderen, bereits auf dem ausgewählten Host bestehenden Containern erstellen. Dies wird erreicht durch Auswahl des gewünschten Containers im Dropdown-Feld "Links". Danach kann mittels des blauen "Plus-Zeichen" die Verknüpfung hinzugefügt werden.

Zum Schluss können noch eine oder mehrere Umgebungsvariablen hinzugefügt werden. Schlüssel und Wert werden in den entsprechenden Feldern eingegeben und mittels Klick auf das blauen "Plus-Zeichen" hinzugefügt.

Mit dem Betätigen der grünen "Hinzufügen"-Taste wird der Container in der Datenbank erfasst und die entsprechenden Aufträge werden erstellt um das gewählte Image auf den Host herunterzuladen und den Container zu erstellten.

Der neue Container erscheint sofort in der Auflistung, bleibt aber, bis er auf dem Host erstellt wurde, im Status "Uknown".

Nachdem die Erstellungsaufträge abgearbeitet wurden, wird der Container zum Status "Stopped" wechseln.

Detailansicht Docker-Container

Mit Klick auf den "Auge-Symbol" gelangt man zur Detailansicht des Containers. Hier kann der Container auch gestartet, gestoppt oder gelöscht werden.

Docker-Container starten

Durch wählen der Funktion 'starten' wird dem Docker-Host ein Auftrag übergeben um den Container zu starten.

Sobald der Container gestartet werden konnte, erscheint er in der Auflistung und Detailansicht mit dem Status "Running".

Docker-Container oder Docker-Host löschen

Mit der Taste "delete" Können Container oder gesamte Hosts gelöscht werden. Diese werden Zur Entfernung markiert und sind, sobald die Löschung erfolgreich durchgeführt wurde, nicht mehr ersichtlich.

Es ist zu beachten, dass bei der Löschung eines gesamten Hosts alle zum Host gehörenden Docker-Container ebenfalls zur Entfernung markiert werden.

Tasks anzeigen

Die Tasks, welche dem Docker-Host übermittelt werden, sind im Untermenü "Tasks" zu finden. Hier sieht der Benutzer den Auftragstyp und ob der Status "Ready", "Sent", "Success" oder "Failed" ist. Die Aufträge können mittels des Filterkästchens nach Wunsch gefiltert werden.

Statusverlauf anzeigen

Unter dem Menüpunkt "Statusverlauf" werden zwei Statusänderungstabellen aufgelistet, eine für die Container und eine für die Hosts. Dem Benutzer ist ersichtlich wie sich der Status verändert hat und zu welcher Zeit. Die Statusverlaufseinträge können mittels des Filterkästchens nach Wunsch gefiltert werden.

Benutzerkonto

Unter dem Benutzerkonto-Menü sind die Kontaktdaten des Benutzers ersichtlich.

Logout

Über den Ausloggen-Knopf oben rechts kann sich der Benutzer von der Applikation ausloggen.
